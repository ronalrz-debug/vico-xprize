## 📌 Project Overview
**vico** is a localized, dynamic geospatial directory designed to map the "invisible economy" within emerging markets. Traditional mapping platforms capture corporate landmarks but overlook micro-businesses, community-trusted services, and informal financial nodes (like local bank correspondents). vico bridges this gap by focusing on raw community utility—allowing users to locate essential services within a 15-minute walking radius.

This project was built and submitted as part of the **XPRIZE Hackathon**.

---

## 🛠️ Built With
* **Frontend:** HTML5, CSS3 (Premium Dark Mode Architecture), Vanilla JavaScript (ES6+).
* **Mapping Engine:** Leaflet.js connected with OpenStreetMap core.
* **Map Stylization:** CartoDB Dark Matter tile rendering.
* **Database:** Lightweight, agile JSON structure optimized for client-side queries.
* **Integrations:** Direct automated WhatsApp communication loops (`wa.me`).

---

## 🚀 Key Features
1. **Intent-Driven Instant Search:** Users search by fluid needs ("copias", "agua", "depósito") rather than corporate brand names.
2. **Proximity-First Layout:** Live mathematical sorting of service points based on immediate user vicinity.
3. **Low-Overhead Infrastructure:** 100% open-source spatial queries running entirely client-side, eliminating server maintenance barriers for marginalized communities.
